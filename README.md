# Blunt_Solver

This code only includes:

2nd order MUSCL: Roe

limiter function: vanleer, minmod, superbee

time integration: TVD RK3, TVD RK2

This code can solve: 

1. supersonic flow past a blunt body

2. other 2d inviscid problems (with some modifications of the code)
